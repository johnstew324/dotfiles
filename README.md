# Dotfiles
Personal dotfiles and workflow for my Ubuntu Distro boot.
Still in progress...



## Installation Instructions
```bash
git clone https://github.com/johnstew324/dotfiles.git ~/dotfiles
cd ~/dotfiles
./install.sh
```

## packages.txt
```bash
packages.txt
```

* Contains a list of packages to be installed via apt ( modify package if needed ).
* Packages includes preinstalled packages on Ubuntu minimal install.
